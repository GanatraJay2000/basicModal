Usage
  1. Download the files
  2. Link the css in the head section
    <link rel="stylesheet" href="modal.css">
    <link rel="stylesheet" href="scrollbar.css" />
  3. Link th JS at the bottom of body
    <script src="modal.js">
  4. Code this in HTML

    For HTML Content in Modal

    • add link/button to open modal
      <button type="button" class="open-modal" data-modal-id="modal1">Open Modal</button>
    • add the modal. Dont forget this lol!
      <div class="custom-modal modal" id="modal1">
           <div class="popup">
               <p> Lorem ipsum </p>
           </div>
      </div>
      you can add anything in the modal.

    For Image Modal

    • add link to open modal
      <a href="#" class="open-image">
           <img class="image" src="image.png" alt="">
      </a>
    • No more steps needed for image modal
Presets

Variables available for changing and their default values

Text Modal

Modal containing HTML content


.custom-modal.modal{
--modal-background: #00000066;
--width: 60%;
--max-height: 80%;
--popup-top: 10%;
--popup-background-color: white;
--popup-background: var(--plain-bg);
--popup-border-width: 6.5px;
--popup-border-color: white;
--popup-border-radius: 5px;
--popup-padding: 20px;
--popup-overflow: hidden scroll;
--popup-shadow: 0px 0px 10px #00000066;
--popup-top: 10%;
--handlecolor: #fa6d8b;
--handlecolorhover: #fc4c72;
--close-color: #888;
--close-hover-color: #000;
}

Image Modal

Modal for Images (useful for galleries)


.custom-modal.gallery{ --modal-background: #00000066;
--width: 60%;
--max-height: 98%;
--popup-top: 10%;
--popup-background-color: transparent;
--popup-background: var(--plain-bg);
--popup-border-width: 0px;
--popup-border-color: white;
--popup-border-radius: 0px;
--popup-padding: 0px;
--popup-overflow: hidden;
--popup-shadow: 0px 0px 0px #00000066;
--popup-img-shadow: 0px 0px 0px #00000066;
--popup-top: 50%;
--close-color: #fff;
--close-hover-color: #000;
}

You can change these values in your custom css on this same class or you can also create a custom class for specific modal and change the variables in its scope.

Examples
Default for HTML Content
For vertically placing the modal, the formula is: value of --height + 2 times --popup-top = 100%
#CSS
.custom-modal.custom-style-1{
--width:30%;
--height:98%;
--popup-padding:30px;
--popup-top:1%;
--handlecolor: #fa6d8b;
--handlecolorhover: #fc4c72;
--close-color: #888;
text-align: justify;
text-align-last: center; // Makes last line centered
}

#HTML
<button type="button" class="open-modal" data-modal-id="modal2">Open Modal</button>

<div class="custom-modal modal custom-style-1" id="modal2">
<div class="popup">
<h2>Some text in the Modal..</h2>
<p>
Lorem ipsum dolor .....
</p>
</div>
</div>

As we are using CSS Variables so we can use regular CSS with it with ease.
If your content is greater than the height of popup you should try
--popup-background:var(--shadow-bg); for more interactivity. 
#CSS
.custom-modal.custom-style-2{ 
--modal-background: #000000dd;
--popup-background-color: rgb(240, 222, 190);
--popup-background: var(--plain-bg);
--popup-border-color: rgb(240, 222, 190);
--width: 30%;
--height: 40%;
--popup-top: 30%;
}

#HTML
<button type="button" class="open-modal" data-modal-id="modal3">Open Modal</button>

<div class="custom-modal modal custom-style-2" id="modal3">
  <div class="popup">
    <h2>Some text in the Modal..</h2>
    <p>
      Lorem ipsum dolor .....
    </p>
  </div>
</div>
